CloudSync Platform

Платформа для синхронизации и управления облачными сервисами с интуитивным интерфейсом и real-time аналитикой.

О проекте

CloudSync — это платформа, позволяющая объединить несколько облачных хранилищ в единый интерфейс. Пользователи могут перемещать файлы между провайдерами, настраивать автоматическую синхронизацию и отслеживать использование дискового пространства.


Проект разработан с использованием React на фронтенде, Node.js + Express на бэкенде и PostgreSQL для хранения данных. Развёрнут в Docker-контейнерах с CI/CD через GitHub Actions.
